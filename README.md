# 3011

**Distributed Systems · AI Agents · Cloud Operations**

Building practical infrastructure for distributed systems, AI agents, and cloud operations.

## Current Focus

- Distributed network measurement and observability
- MCP servers and dependable agent tooling
- Kubernetes operations and incident diagnosis
- Self-hosted infrastructure and automation

## Featured Projects

| Project | Description |
| --- | --- |
| [**CF Scanner**](https://github.com/3011/cfscan) | Distributed Cloudflare Anycast IP scanning, ranking, automation, and monitoring platform. |
| [**Obsidian Vault MCP**](https://github.com/3011/obsidian-vault-mcp) | Headless MCP server for safely operating Markdown and Obsidian vaults. |
| [**exec-mcp**](https://github.com/3011/exec-mcp) | Bounded SSH remote execution and file-transfer gateway for MCP clients. |
| [**AI Ops**](https://github.com/3011/ai-ops) | Evidence-driven Kubernetes incident diagnosis and AIOps workflow console. |

## Engineering Principles

Small interfaces. Explicit boundaries. Observable behavior. Operable systems.

# 中文

专注于分布式系统、AI Agent 基础设施与云原生运维。

## 当前方向

- 分布式网络测量与可观测性
- MCP Server 与可靠的 Agent 工具链
- Kubernetes 运维与故障诊断
- 自托管基础设施与自动化

| 项目 | 简介 |
| --- | --- |

保持接口精简、边界明确、行为可观测，并让系统真正可运维。
